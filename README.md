**BeautyCorner Mirror Tool** is a lightweight Blender add-on that simplifies the creation of mirrored cuts at a specified angle.  
The plugin automatically creates an `Empty`, adds a `Mirror` modifier, configures it, and allows you to toggle the **Flip Y** parameter with a single click.

- Apply the modifier and automatically remove the helper `Empty`
